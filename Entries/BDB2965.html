II.
of God: kindness, lovingkindness in condescending to the needs of his creatures. He is חַסְדָּם their goodness favour Jonah 2:9 חַסְדִּי Ps 144:2 אֱלֹהֵי חַסְדִּי God of my kindness Ps 59:18 in Ps 59:11 read אֱלֹהַי חַסְדּוֺ my God with his kindness 𝔊 ᵑ9 Ew Hup De Pe Che Bae; his is the kindness Ps 62:13 it is with him Ps 130:7 he delights in it Mic 7:18.

a. in redemption from enemies and troubles Gen 19:19 ; Gen 39:21 ( J ), Exod 15:13 (song), Jer 31:3; Ezra 7:28; Ezra 9:9; Ps 21:8 ; Ps 31:17 ; Ps 31:22 ; Ps 32:10 ; Ps 33:22 ; Ps 36:8 ; Ps 36:11 ; Ps 42:9; Ps 44:27 ; Ps 48:10 ; Ps 59:17 ; Ps 66:20 ; Ps 85:8; Ps 90:14 ; Ps 94:18 ; Ps 107:8 ; Ps 107:15 ; Ps 107:21 ; Ps 107:31 ; Ps 143:8 ; Ps 143:12 ; Job 37:13 ; Ruth 1:8; Ruth 2:20 men should trust in it Ps 13:6 ; Ps 52:10 rejoice in it Ps 31:8 hope in it Ps 33:18; Ps 147:11.

c. in quickening of spiritual life Ps 109:26 ; Ps 119:41 ; Ps 119:76 ; Ps 119:88 ; Ps 119:124 ; Ps 119:149; Ps 119:159.
d. in redemption from sin Ps 25:7; Ps 51:3.
e. in keeping the covenants, with Abraham Mic 7:20 with Moses and Israel שׁמר הַבְּרִית וְ(הַ)חֶסֶד keep-eth the covenant and the lovingkindness Deut 7:9; Deut 7:12; 1Kgs 8:23 = 2Chr 6:14 ; Neh 1:5; Neh 9:32; Dan 9:4 with David and his dynasty 2Sam 7:15 = 1Chr 17:13 ; 2Sam 22:51 = Ps 18:51 , 1Kgs 3:6 (twice in verse) = 2Chr 1:8 ; Ps 89:29 ; Ps 89:34 with the wife Zion Isa 54:10.
