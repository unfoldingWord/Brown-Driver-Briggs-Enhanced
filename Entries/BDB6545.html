BDB6545 [H6316]

Biblical Hebrew

פּוּט proper name [of a people] probably Libyans, or Libyan tribe; usually named with African peoples: Nah 3:9 ; Jer 46:9; Ezek 27:10; Ezek 30:5; Ezek 38:5 ( 𝔊 usually Λίβυες); Gen 10:6 ( P ) = 1Chr 1:8 (Φουδ); + Isa 66:19 (for ᵑ0 פּוּל; 𝔊 φουδ), — see Di Gen 10:6 JenZA x. 325 ff.
