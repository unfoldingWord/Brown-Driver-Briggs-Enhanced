Biblical Hebrew

אוֺת noun masculine79 Exod 4:8 (feminine Josh 24:17 ) sign (Arabic , plural , Aramaic אָתָא, ) — ׳ א Gen 4:15 +; construct Gen 9:12 +; plural אוֺתוֺת Exod 4:9 + etc. —

1. sign, pledge, token Gen 4:15 ( J ); אוֺת אֱמֶת true, token Josh 2:12 ( J ); of blood of passover Exod 12:13 ( P ); אוֺת לְטוֺבָה token for good Ps 86:17 pledges, assurances of travellers Job 21:29.
2. signs, omens promised by prophets as pledges of certain predicted events 1Sam 10:7; 1Sam 10:9 + 1Sam 10:1 where ׳ א insert ᵐ5 ᵑ9 , see We Dr; especially phrase זֶה הָאוֺת לְ Exod 3:12; 1Sam 2:34; 1Sam 14:10; 2Kgs 19:29 ; Jer 44:29; Isa 7:11; Isa 7:14 probably also Isa 44:25 (of false prophets).
3. sign, symbol of prophets Isa 8:18 compare Ezek 4:3.
4. signs, miracles, as pledges or attestations of divine presence & interposition Exod 4:8 (twice in verse); Exod 4:9 ( J ) Exod 7:3 ( P ) Exod 8:19 ( J ) Ps 74:9 ; 2Kgs 19:29; 2Kgs 20:8; 2Kgs 20:9 = Isa 37:30; Isa 38:7; Isa 38:22 with עָשָׂה Exod 4:17; Exod 4:30; Num 14:11; Num 14:22; Josh 24:17 (all J E ) Deut 11:3 ; Judg 6:17 with צִוָּה Exod 4:28 ( J ); with שִׁית Exod 10:1 ( J ); with שִׂים Exod 10:2 ( J ) Ps 78:43; Isa 66:19 הָאוֺת וְהַמּוֺפֵת (see מוֺפֵת ) Deut 13:3 compare Deut 28:46; Isa 20:3 ׳ אוֺ מ ׳ א Deut 13:2 אוֺתוֺת וּמוֺפְתִים Deut 4:34; Deut 7:19; Deut 26:8; Deut 29:2; Jer 32:31 with נָתַן Deut 6:22; Neh 9:10 with שִׂים Jer 32:20; Ps 105:27 with שָׁלַח Deut 34:11; Ps 135:9.
5. signs, memorials, stones from Jordan Josh 4:6 ( J ); metal of censers Num 17:3 ( P ); Aaron's rod Num 17:25 ( P ); עוֺלָם ׳ א Isa 55:13 probably also Ezek 14:8 (|| מָשָׁל); signs on hands, etc., Exod 13:9; Exod 13:16 ( J ) = Deut 6:8 ; Deut 11:18, probably belong here; also memorial pillar in Egypt Isa 19:20.
6. sign, pledge of covenant, הַבְּרִית ׳ א (see ברית) e.g. rainbow, of Noachian convenant Gen 9:12; Gen 9:13; Gen 9:17 ( P ); circumcision, of Abrahamic covenant Gen 17:11 ( P ); the sabbath Exod 31:13; Exod 31:17 ( P ); Ezek 20:12; Ezek 20:20.
7. ensigns, standards Num 2:2 ( P ) Ps 74:4 .
8. signs, tokens of changes of weather & times Gen 1:14 ( P ; of heavenly luminaries) אוֺתוֺת הַשָּׁמַיִם Jer 10:2 (changes of the heavens as omens to frighten the nations) compare Ps 65:9.
