Biblical Hebrew

חָרָה verb burn, be kindled, of anger (Aramaic חרי Pa`el cause, fire to burn (rare); Zinjirli חרא anger, NöZMG 1893, 98. 103; Arabic burning sensation, in throat, etc., from rage and pain) —

Qal Perfect ׳ ח Gen 4:6 + 20 t.; Imperfect יֶחֱרֶה Exod 32:11 + 2 t.; יִ֫חַר Gen 18:30 + 7 t.; וַיִּ֫חַר Gen 4:5 + 46 t.; Infinitive absolute חָרֹה 1Sam 20:7 construct חֲרוֺת 2Sam 24:1; Ps 124:3 —
1. of man:
a. חרה אף (one's) anger was kindled, burned Gen 39:19 ; Num 22:27 ( J ), Exod 32:19; Exod 32:22 ( J E ), Judg 9:30 ; Judg 14:19 ; 1Sam 11:6 ; Job 32:2; Job 32:5 with בְּ against Gen 30:2 ( E ), Gen 44:18 ( J ), 1Sam 17:28; 1Sam 20:30; 2Sam 12:5; 2Chr 25:10; Ps 124:3; Job 32:2; Job 32:3 with אֶל Num 24:10 ( E ).
b. impersonal, אף omitted; חרה לְ it was kindled for (him) he burned with anger Gen 31:36 ( E ), Gen 4:5 ; Gen 4:6; Gen 34:7; Num 16:15 ( J ), 1Sam 15:11 (read perhaps וַיֵּצֶר, Weir in Dr ), 1Sam 18:8 ; 1Sam 20:7 (twice in verse); 2Sam 3:8 ; 2Sam 6:8 = 1Chr 13:11 ; 2Sam 13:21 ; 2Sam 19:43 ; Neh 3:33; Neh 4:1; Neh 5:6; Jonah 4:1; Jonah 4:4; Jonah 4:9 (twice in verse); insert ויחר לדוד 2Sam 11:22 𝔊 We Dr; בְּעֵינֵי ׳ ח Gen 31:35 ; Gen 45:5 ( E ).
2. of God:
a. אף ׳ ח Exod 22:23 ; Num 22:22 ( E ), Num 11:1 ; Num 11:10 ; Num 32:10 ( J ); with בְּ Exod 4:14; Exod 32:10 ; Num 12:9 ; Deut 31:17 ( J ), Deut 6:15 ; Deut 7:4 ; Deut 11:17; Deut 29:26; Josh 23:16 ( D ), Judg 6:39 ; 2Sam 6:7 = 1Chr 13:10 ; Hos 8:5; 2Kgs 23:26 ; 2Chr 25:15 ; Job 42:7 בְּעַם Exod 32:11 ; Num 11:33 ( J ), Isa 5:25 ; Ps 106:40 בישׂראל Num 25:3; Num 32:13 ( J E ); Judg 2:14 ; Judg 2:20 ; Judg 3:8; Judg 10:7; 2Sam 24:1 ; 2Kgs 13:3 בבני ישׂראל Josh 7:1 ( R ); בנהרים, אף omitted, Hab 3:8 with עַל Zech 10:3 .
b. חרה לְ Gen 18:30 ; Gen 18:32 ( J ) 2Sam 22:8 = Ps 18:8 . —
Note: חרה not in HP (who use קצף); nor in Jeremiah, Ezekiel, Isa2, Proverbs or poetry of Job.
